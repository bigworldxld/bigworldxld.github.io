一、OpenClaw 简介

OpenClaw（小龙虾）是一款可本地部署、支持多平台接入的 AI 助手工具。它不仅能跟你对话，还能直接操控电脑、自动办公、整理文件、操作浏览器、对接飞书/钉钉，相当于你电脑里 24 小时待命的私人助理。

主要优势

  1. 本地部署：数据不泄露，隐私更安全
  2. 多平台支持：支持飞书、微信、QQ、Discord 等
  3. 自动化能力：能直接操作你的电脑，帮你整理文件、写文档、搜资料、发消息、控制浏览器
  4. 可扩展性：支持插件系统，可自由扩展功能

适用场景

  • 日常自动化工作
  • 交流沟通
  • 本地系统任务处理
  • 文档编写与整理
  • 浏览器自动化操作

二、OpenClaw CLI 指令

环境准备

  1. 检查 Node.js 版本：需要 Node.js 18+，推荐使用 LTS 版本
  2. 安装 Git：用于克隆仓库和管理版本

安装指令

# 检查 Node.js 版本
node -v

# 安装 CLI 工具
npm install -g openclaw

# 验证安装
openclaw --version

安装注意事项

  • 如果安装失败，可能需要使用管理员权限运行 PowerShell
  • 可以使用 npm install -g openclaw --force 强制安装
  • 安装完成后，可能需要重启 PowerShell 才能使用 openclaw 命令

常用指令

# 启动网关
openclaw gateway start

# 停止网关
openclaw gateway stop

# 重启网关
openclaw gateway restart

# 运行配置向导
openclaw wizard

# 安装技能
openclaw skill install <skill-name>

# 列出已安装技能
openclaw skill list

# 更新技能
openclaw skill update <skill-name>

# 卸载技能
openclaw skill uninstall <skill-name>

# 搜索技能
openclaw skill search <keyword>

# 查看版本
openclaw --version

# 查看帮助
openclaw --help

指令说明

  • gateway：管理 OpenClaw 网关服务
  • wizard：运行配置向导，帮助你快速配置 OpenClaw
  • skill：管理 OpenClaw 技能
  • –version：查看 OpenClaw 版本
  • –help：查看帮助信息

三、ClawHub Skills 市场

ClawHub 是 OpenClaw 的技能市场，提供各种预定义的自动化技能。你可以在这里找到各种实用的技能，也可以上传自己开发的技能。

主要特点

  • 生态丰富: 可从 ClawHub 安装第三方 Skills，涵盖办公、开发、娱乐等多个领域
  • 自定义开发: 支持编写自定义 Skills 扩展能力，可以使用 YAML 或 JavaScript 开发
  • 典型 Skills: GitHub、Weather、Coding Agent、PDF 处理、文件管理等
  • 安全可靠: 所有技能都经过审核，确保安全可靠

常用技能推荐

  • @openclaw/pdf-tools: PDF 高级处理（合并、拆分、加水印）
  • @openclaw/wechat: 微信集成插件（需额外配置）
  • @openclaw/qq: QQ 机器人集成插件
  • @openclaw/file-manager: 文件管理自动化
  • @openclaw/browser-automation: 浏览器自动化操作

使用方法

# 搜索技能
openclaw skill search <keyword>

# 安装技能
openclaw skill install <skill-name>

# 更新技能
openclaw skill update <skill-name>

# 卸载技能
openclaw skill uninstall <skill-name>

# 列出已安装技能
openclaw skill list

技能开发

如果你想开发自己的技能，可以参考官方文档，使用 YAML 或 JavaScript 编写技能。开发完成后，可以上传到 ClawHub 分享给其他用户。

四、接入飞书

第一步: 创建飞书应用

  1. 打开飞书开放平台：访问 飞书开放平台，使用飞书账号登录
  2. 创建新应用：点击“创建企业自建应用”，填写应用名称、描述等信息
  3. 配置应用权限：在“权限管理”中添加所需权限，如“消息发送”、“群组管理”等
  4. 获取应用凭证：在“凭证与基础信息”中获取 app_id 和 app_secret

第二步: 配置 OpenClaw

// openclaw.json 配置
{
  "channels": {
    "feishu": {
      "enabled": true,
      "appId": "your-app-id",
      "appSecret": "your-app-secret",
      "connectionMode": "websocket",
      "domain": "feishu",
      "groupPolicy": "open",
      "messagePolicy": "all",
      "reactionPolicy": "allow"
    }
  }
}

第三步: 启动服务

openclaw gateway start

第四步: 测试连接

  1. 在飞书群中 @ 你的机器人
  2. 发送消息，如“你好”
  3. 检查机器人是否回复

飞书配置说明

  • appId/appSecret：飞书应用凭证，用于身份验证
  • connectionMode：连接模式，支持 websocket 和 webhook
  • groupPolicy：群聊策略，open 表示允许所有群聊
  • messagePolicy：消息策略，all 表示接收所有消息
  • reactionPolicy：反应策略，allow 表示允许使用表情反应

常见问题

  • 机器人不回复：检查 app_id 和 app_secret 是否正确，权限是否配置完整
  • 无法接收消息：检查 connectionMode 是否正确，网络连接是否正常
  • 消息发送失败：检查消息内容是否符合飞书规范，是否包含敏感信息

五、OpenClaw 面板功能

OpenClaw 提供了一个直观的 Web 面板，用于管理和配置 AI 助手。你可以通过浏览器访问面板，进行各种设置和操作。

1. Gateway 面板

  • 管理 HTTP 服务：启动、停止、重启网关服务
  • 配置端口和绑定地址：设置网关监听的端口和地址
  • 查看连接状态：实时查看网关的连接状态和性能指标
  • 日志管理：查看网关的日志信息，方便排查问题

2. Skills 面板

  • 管理已安装的技能：查看、安装、更新、卸载技能
  • 启用/禁用技能：根据需要启用或禁用特定技能
  • 查看技能详细信息：查看技能的描述、版本、作者等信息
  • 技能配置：配置技能的参数和设置

3. Sessions 面板

  • 查看会话历史：查看所有会话的历史记录
  • 管理会话状态：启动、停止、重启会话
  • 查看会话日志：查看会话的详细日志信息
  • 会话统计：查看会话的统计信息，如响应时间、成功率等

4. Models 面板

  • 配置 AI 模型：添加、删除、配置 AI 模型
  • 切换默认模型：设置默认使用的 AI 模型
  • 查看模型状态：实时查看模型的使用状态和性能指标
  • 模型测试：测试模型的响应和性能

5. Channels 面板

  • 管理集成渠道：添加、删除、配置集成渠道（飞书、微信、QQ 等）
  • 配置渠道参数：设置渠道的参数和权限
  • 查看连接状态：实时查看渠道的连接状态
  • 渠道日志：查看渠道的日志信息

6. Settings 面板

  • 全局配置：设置 OpenClaw 的全局参数和选项
  • 安全设置：配置安全策略和权限
  • 通知设置：配置通知方式和接收人
  • 备份与恢复：备份和恢复配置文件

六、常见问题

1. API 速率限制错误

当出现 ⚠️ API rate limit reached. Please try again later. 报错时，停止执行任务，并返回信息提示：”API速率限制已达到，请稍后再试。”

2. 飞书连接失败

  • 检查飞书应用配置
  • 确保网络连接正常
  • 查看网关日志

3. 技能安装失败

  • 检查网络连接
  • 确保技能名称正确
  • 查看技能市场状态

4. 插件加载失败

  • 检查插件配置文件
  • 确保插件依赖已安装
  • 查看插件日志

5. 网关启动失败

  • 检查端口是否被占用
  • 确保配置文件格式正确
  • 查看网关日志

七、高级功能

1. 插件系统

OpenClaw 插件系统允许你扩展功能，无论是官方插件还是自定义插件，都能通过标准化接口与核心系统交互。插件是 OpenClaw 生态系统的重要组成部分，允许你根据自己的需求定制功能。

常用官方插件

  • @openclaw/pdf-tools: PDF 高级处理（合并、拆分、加水印、加密解密）
  • @openclaw/wechat: 微信集成插件（需额外配置，支持私聊和群聊）
  • @openclaw/qq: QQ 机器人集成插件（支持私聊和群聊）
  • @openclaw/file-manager: 文件管理自动化（批量重命名、移动、删除）
  • @openclaw/browser-automation: 浏览器自动化操作（网页截图、表单填写、数据抓取）
  • @openclaw/calendar: 日历集成插件（支持飞书日历、Outlook 日历）
  • @openclaw/email: 电子邮件集成插件（支持 SMTP、POP3、IMAP）

自定义插件开发

你可以使用 YAML 或 JavaScript 编写专属插件，实现个性化需求：

  • 自定义文件命名规则
  • 专属工作流
  • 个性化功能
  • 第三方服务集成

插件开发步骤

  1. 创建插件目录
  2. 编写插件配置文件（plugin.yaml）
  3. 编写插件代码（index.js 或 main.js）
  4. 测试插件
  5. 上传到 ClawHub 分享给其他用户

2. 源码定制

通过 Fork 官方仓库，可自由修改代码、定制功能、优化配置，后续还能将自己的改进贡献回社区。

源码定制步骤

  1. Fork 官方仓库
  2. 克隆到本地
  3. 修改代码
  4. 测试修改
  5. 提交 PR 贡献回社区

3. 服务器部署

想要拥有一个属于自己的 7*24 小时运行的 AI 助手，需要一台服务器：

硬件要求

  • 最低配置：2GB 内存，1 核 CPU，20GB 硬盘
  • 推荐配置：4GB 内存，2 核 CPU，50GB 硬盘

软件要求

  • Node.js 18+（推荐 LTS 版本）
  • Git
  • PM2（用于进程管理）

部署步骤

  1. 安装 Node.js 和 Git
  2. 克隆 OpenClaw 仓库
  3. 安装依赖
  4. 配置 OpenClaw
  5. 使用 PM2 启动服务
  6. 配置反向代理（可选）

4. 记忆系统

OpenClaw 拥有强大的记忆系统，可以帮助你记住重要的信息和上下文。

记忆类型

  • 短期记忆：会话期间的上下文信息
  • 长期记忆：通过 MEMORY.md 文件持久化保存的信息
  • 技能记忆：技能使用的历史记录

记忆管理

  • 可以通过面板管理记忆
  • 可以通过 API 访问记忆
  • 可以通过技能扩展记忆功能

九、常见问题

1. 安装失败

  • 问题：npm install -g openclaw 失败
  • 解决方案：使用管理员权限运行 PowerShell，或者使用 –force 参数强制安装

2. 网关无法启动

  • 问题：openclaw gateway start 失败
  • 解决方案：检查端口是否被占用，配置文件是否正确

3. 飞书机器人不回复

  • 问题：飞书群中 @ 机器人但没有回复
  • 解决方案：检查 app_id 和 app_secret 是否正确，权限是否配置完整

4. 技能安装失败

  • 问题：openclaw skill install 失败
  • 解决方案：检查网络连接，技能名称是否正确
