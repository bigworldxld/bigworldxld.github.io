视频异常检测工具 - 打包可执行文件指南

📦 打包说明

本工具提供将Python脚本打包成独立可执行文件的功能，无需安装Python环境即可运行。

🎯 快速开始

macOS/Linux 打包

cd skills/video-anomaly-detector-1.0.0
./build.sh

#!/bin/bash
"""
视频异常检测工具打包脚本（macOS/Linux）
生成独立可执行文件 video_detect
"""

set -e  # 遇到错误立即退出

echo "🎬 视频异常检测工具打包脚本"
echo "========================================"
echo ""

# 检查Python版本
echo "📋 检查环境..."
python3 --version || { echo "❌ 未找到Python3"; exit 1; }

# 检查PyInstaller
if ! python3 -c "import PyInstaller" 2>/dev/null; then
    echo "⚠️  PyInstaller未安装，正在安装..."
    pip3 install --break-system-packages pyinstaller
fi

# 检查依赖
echo "📦 检查依赖包..."
for package in opencv-python numpy pandas openpyxl; do
    if ! python3 -c "import ${package//-/_}" 2>/dev/null; then
        echo "⚠️  $package 未安装，正在安装..."
        pip3 install --break-system-packages $package
    fi
done

echo ""
echo "🔨 开始打包..."
echo "========================================"

# 清理旧的构建
if [ -d "build" ]; then
    echo "🗑️  清理旧的构建目录..."
    rm -rf build
fi

if [ -d "dist" ]; then
    echo "🗑️  清理旧的发布目录..."
    rm -rf dist
fi

# 使用PyInstaller打包
echo "📦 正在打包（这可能需要几分钟）..."
pyinstaller video_detect.spec

# 检查是否成功
if [ -f "dist/video_detect" ]; then
    echo ""
    echo "✅ 打包成功！"
    echo "========================================"
    echo "📁 可执行文件位置: dist/video_detect"
    echo ""
    echo "📊 文件大小:"
    du -h dist/video_detect
    echo ""
    echo "🚀 使用方法:"
    echo "   ./dist/video_detect video.mp4"
    echo "   ./dist/video_detect data.xlsx --compare"
    echo ""
    echo "💡 移动到系统路径（可选）:"
    echo "   sudo cp dist/video_detect /usr/local/bin/"
    echo "   video_detect --help"
    echo ""
else
    echo "❌ 打包失败，请检查错误信息"
    exit 1
fi

Windows 打包

cd skills\video-anomaly-detector-1.0.0
build.bat

@echo off
REM 视频异常检测工具打包脚本（Windows）
REM 生成独立可执行文件 video_detect.exe

echo 🎬 视频异常检测工具打包脚本
echo ========================================
echo.

REM 检查Python版本
echo 📋 检查环境...
python --version >nul 2>&1
if errorlevel 1 (
    echo ❌ 未找到Python
    exit /b 1
)

REM 检查PyInstaller
python -c "import PyInstaller" >nul 2>&1
if errorlevel 1 (
    echo ⚠️  PyInstaller未安装，正在安装...
    pip install pyinstaller
)

REM 检查依赖
echo 📦 检查依赖包...
python -c "import cv2" >nul 2>&1
if errorlevel 1 (
    echo ⚠️  opencv-python 未安装，正在安装...
    pip install opencv-python
)

python -c "import numpy" >nul 2>&1
if errorlevel 1 (
    echo ⚠️  numpy 未安装，正在安装...
    pip install numpy
)

python -c "import pandas" >nul 2>&1
if errorlevel 1 (
    echo ⚠️  pandas 未安装，正在安装...
    pip install pandas
)

python -c "import openpyxl" >nul 2>&1
if errorlevel 1 (
    echo ⚠️  openpyxl 未安装，正在安装...
    pip install openpyxl
)

echo.
echo 🔨 开始打包...
echo ========================================

REM 清理旧的构建
if exist build (
    echo 🗑️  清理旧的构建目录...
    rmdir /s /q build
)

if exist dist (
    echo 🗑️  清理旧的发布目录...
    rmdir /s /q dist
)

REM 使用PyInstaller打包
echo 📦 正在打包（这可能需要几分钟）...
pyinstaller video_detect.spec

REM 检查是否成功
if exist "dist\video_detect.exe" (
    echo.
    echo ✅ 打包成功！
    echo ========================================
    echo 📁 可执行文件位置: dist\video_detect.exe
    echo.
    echo 🚀 使用方法:
    echo    dist\video_detect.exe video.mp4
    echo    dist\video_detect.exe data.xlsx --compare
    echo.
    echo 💡 添加到系统路径（可选）:
    echo    将 dist\video_detect.exe 复制到 C:\Windows\System32
    echo    或者添加 dist 目录到 PATH 环境变量
    echo.
) else (
    echo ❌ 打包失败，请检查错误信息
    exit /b 1
)

📋 前置要求

1. Python 环境

  • Python 3.8+ （推荐 3.10+）
  • pip（Python包管理器）

2. 依赖包

打包脚本会自动安装以下依赖：

  • pyinstaller - 打包工具
  • opencv-python - 视频处理
  • numpy - 数值计算
  • pandas - Excel处理
  • openpyxl - Excel读写

3. 系统依赖

macOS:

brew install ffmpeg

Linux (Ubuntu/Debian):

sudo apt-get install ffmpeg

Windows:

  • 下载 ffmpeg
  • 添加到系统PATH
# -*- mode: python ; coding: utf-8 -*-
"""
PyInstaller 配置文件
用于将 video_detect 打包成独立可执行文件

修复OpenCV递归导入问题
"""
import sys
from pathlib import Path

block_cipher = None

# 获取cv2的数据文件路径（Haar级联分类器等）
import cv2
cv2_path = Path(cv2.__file__).parent
cv2_data_path = cv2_path / 'data'

a = Analysis(
    ['scripts/detect.py'],
    pathex=[],
    binaries=[],
    datas=[
        ('scripts/audio_detector.py', 'scripts'),
        # 添加OpenCV的数据文件（如Haar级联分类器）
        (str(cv2_data_path), 'cv2/data'),
    ],
    hiddenimports=[
        # OpenCV相关
        'cv2',
        'cv2.cv2',  # 重要：明确包含cv2.cv2避免递归
        # NumPy相关
        'numpy',
        'numpy.core._multiarray_umath',
        'numpy.core._multiarray_tests',
        # Pandas相关
        'pandas',
        'pandas._libs.tslibs.timedeltas',
        # Excel支持
        'openpyxl',
        'openpyxl.cell._writer',
    ],
    hookspath=[],
    hooksconfig={},
    runtime_hooks=[],
    excludes=[
        # 排除不需要的大型模块
        'tkinter',
        'matplotlib',
        'PIL',
        'scipy',
    ],
    win_no_prefer_redirects=False,
    win_private_assemblies=False,
    cipher=block_cipher,
    noarchive=False,
)

pyz = PYZ(a.pure, a.zipped_data, cipher=block_cipher)

exe = EXE(
    pyz,
    a.scripts,
    a.binaries,
    a.zipfiles,
    a.datas,
    [],
    name='video_detect',
    debug=False,
    bootloader_ignore_signals=False,
    strip=False,
    upx=True,
    upx_exclude=[],
    runtime_tmpdir=None,
    console=True,
    disable_windowed_traceback=False,
    argv_emulation=False,
    target_arch=None,
    codesign_identity=None,
    entitlements_file=None,
)

🚀 使用可执行文件

基本用法

macOS/Linux:

# 检测单个视频
./dist/video_detect video.mp4

# 检测URL视频
./dist/video_detect https://example.com/video.mp4

# Excel批量检测
./dist/video_detect data.xlsx --limit 10

# 对比检测
./dist/video_detect transcoded.mp4 --original original.mp4

Windows:

# 检测单个视频
dist\video_detect.exe video.mp4

# 检测URL视频
dist\video_detect.exe https://example.com/video.mp4

# Excel批量检测
dist\video_detect.exe data.xlsx --limit 10

# 对比检测
dist\video_detect.exe transcoded.mp4 --original original.mp4

高级用法

# 快速检测（每30帧采样）
./dist/video_detect video.mp4 -s 30

# 只检测关键问题
./dist/video_detect video.mp4 -t freeze,decode_error

# Excel批量对比（自动下载原片）
./dist/video_detect data.xlsx --compare --start 0 --stop 10 -s 15

# 保存批量报告
./dist/video_detect data.xlsx --batch-output report.json

💡 安装到系统路径

macOS/Linux

将可执行文件复制到系统路径：

# 方法1：复制到 /usr/local/bin
sudo cp dist/video_detect /usr/local/bin/

# 方法2：创建软链接
sudo ln -s $(pwd)/dist/video_detect /usr/local/bin/video_detect

# 验证安装
video_detect --help

Windows

方法1：复制到系统目录

copy dist\video_detect.exe C:\Windows\System32\

方法2：添加到PATH环境变量

  1. 右键”此电脑” → 属性 → 高级系统设置
  2. 环境变量 → 系统变量 → Path → 编辑
  3. 新建 → 输入 dist 目录的完整路径
  4. 确定

验证安装：

video_detect --help

🔍 故障排除

问题1：打包失败

现象：

ModuleNotFoundError: No module named 'cv2'

解决：

pip3 install opencv-python numpy pandas openpyxl

问题2：可执行文件太大

现象：

dist/video_detect: 300MB+

解决：

# 使用 --exclude-module 排除不需要的模块
# 编辑 video_detect.spec，在 excludes 中添加：
excludes=['tkinter', 'matplotlib', 'PIL'],

问题3：macOS运行被阻止

现象：

"video_detect" cannot be opened because the developer cannot be verified.

解决：

# 允许运行
xattr -cr dist/video_detect

# 或者在系统偏好设置 → 安全性与隐私 → 通用 → 点击"仍要打开"

问题4：Windows运行被阻止

现象：

Windows protected your PC

解决：

  • 点击”更多信息”
  • 点击”仍要运行”

问题5：ffmpeg找不到

现象：

❌ ffmpeg命令不存在

解决：

macOS:

brew install ffmpeg

Linux:

sudo apt-get install ffmpeg

Windows:

  • 下载 ffmpeg
  • 解压到 C:\ffmpeg
  • 添加 C:\ffmpeg\bin 到PATH

🎉 优势

1. 独立运行

  • ✅ 无需安装Python环境
  • ✅ 无需安装依赖包
  • ✅ 直接运行，开箱即用

2. 跨平台

  • ✅ macOS（Intel + Apple Silicon）
  • ✅ Linux（x86_64）
  • ✅ Windows（x64）

3. 便于分发

  • ✅ 单文件分发
  • ✅ 无需配置环境
  • ✅ 团队协作方便

📝 注意事项

1. 文件大小

  • 可执行文件约150-160MB
  • 包含Python运行时和所有依赖
  • 首次加载可能较慢（2-3秒）

2. 更新

  • 脚本更新后需要重新打包
  • 建议保留源代码以便更新

3. 安全性

  • macOS和Windows可能需要手动允许运行
  • 建议对exe文件进行代码签名（企业分发）
