视频播放器合集

PotPlayer（只支持windows，windows直接用这个就完全ok）

PotPlayer 的定位就是：Windows 平台上的全能本地播放器。

PotPlayer是一款功能强大的多媒体播放器，由韩国的Daum通信公司开发。它支持在Windows操作系统上播放各种音频和视频文件。 PotPlayer支持常见的视频格式（如MP4、AVI、MKV等）和音频格式（如MP3、FLAC、WAV等）。它还支持高清视频播放，并具有流畅的播放性能和优秀的画质。

PotPlayer的界面简洁明了，易于使用。它提供了各种常用的播放控制选项，如播放、暂停、快进、快退等。此外，PotPlayer还提供了一些高级功能，如调整音频和视频设置、添加字幕、截图等。

PotPlayer还具有强大的自定义功能。用户可以根据自己的喜好设置播放器的外观和行为。例如，可以更改皮肤、调整字体大小、设置快捷键等。

KMPlayer（windows电脑端+安卓端）

KMPlayer中文版是基于kmplayer 官方版基础上，通过增加第三方插件、解码器实现的增强版，能对各种流行视频、HDTV 实现更佳的支持和回放。本软件的出发点是对系统进行很少改动的前提下打造全能影音播放器，所以增加的第三方插件绝大部分是注册到 kmplayer，而不是系统里。推荐安装环境是 Windows XP/Windows 2003/Vista/Windows 7。

KMPlayer特色功能

支持全部音视频格式，一站式服务

KMPlayer支持几乎全部音视频格式，主流视频包括：AVI、RealMedia、MPEG 1/2/4、ASF、MKV、FLV、DVD、MP4、Xvid、DivX、H.264等。主流音频格式包括：APE、MP3、WAV、MPC、Flac、MIDI等。

智能音场平衡技术，音效更显震撼

KMPlaye通过音效控制面板、可在不同声效中进行选择和切换、更可对音速快慢进行调节、完全实现个性化定制、享受非凡音效。

图像增强引擎，高品质观影体验

4倍速驱动、降低画面噪点、锐利画质呈现、即便低画质视频也可照样全屏播放。用户还可通过视频控制面板画质指标进行调节、显著提升视觉体验。

更专业的控制面板

KMPlayer提供更专业的控制面板、无论您想对视频进行任何调整，控制面板都可以帮您实现。亮度、对比度、饱和度、启用锐化、模糊、降噪等插件、调整屏幕大小、拉伸、全屏。

快捷的操作方式

KMPlaye提供了适合中国用户使用习惯的快捷操作方式、无论是“小键盘区”、“鼠标”、“功能键区”还是“主键盘区”、我们都可以找到对应的操作。

特色的双字幕功能

字幕对欣赏电影尤为重要、KMPlaye是一款具有更强大的字幕功能的播放器。KMPlayer只需简单设置、双字幕便能让您获得您想要的效果。

VLC Media Player（Windows、Mac、Linux、Android）

今天要给大家介绍的软件简直是软件领域的“全能战士”——VLC！不要被它“播放器”这个名字给迷惑了，它可不仅仅是个播放器，简直就是个软件的变形金刚！

VLC是一款强大的多媒体播放器，支持几乎所有主流的音频和视频格式，无需担心因为格式问题而无法播放文件。包括但不限于：

  • 视频：MPEG, DIVX, H.264, MKV, WebM, WMV, MP4, AVI等。

  • 音频：MP3, AAC, FLAC, OGG, WMA等。

  • 流媒体协议：HTTP, RTSP, RTP, MMS, FTP等。

除了以上功能，VLC是跨平台的，可在Windows、Mac、Linux、Android以及多种移动设备上运行，确保用户在不同设备上的无缝体验。 还支持直播、流媒体播放、本地播放、录制、转码、甚至可以作为服务器使用，是一款真正的多媒体“全能选手”。说它是播放器，简直是侮辱了它的多才多艺！

打开VLC，就像打开了软件的新大门，友好的界面，简单的操作，哪怕你是个“技术小白”，也能在它的陪伴下自信满满！

想看电影、听音乐？VLC搞定！想看在线直播？VLC搞定！想把一种格式的文件转成另一种格式？VLC也搞定！简直是软件领域的“随心所欲”。

别以为全能就会慢，VLC的速度可是杠杠的，播放高清视频毫不费力，而且还不卡顿。速度和性能都是它的“拿手好戏”。

VLC的团队可是认真的，不仅经常更新，而且有问题了也能及时修复。感觉自己的建议被采纳，就像是参与了软件的创作！

这一点简直是VLC的“骄傲”，它是开源软件，用户可以免费使用，而且开源的特性也意味着用户可以查看和修改源代码，不花一分冤枉钱，享受这么多功能简直是软件史上的一大幸事！

VLC对你的文件保护得比女神还严密，不怕文件泄露，不怕隐私曝光，安全放心！

而与商业软件相比，它的界面相对简单：尽管VLC的界面足够直观，但相比一些商业软件，它可能显得简单一些，缺少一些炫酷的设计。对于一些新手用户来说，VLC的高级功能和设置可能会显得复杂，一些简单的操作可能需要花一些时间适应。有时候更新速度可能相对较慢，可能不能及时适应新的多媒体格式或者操作系统的更新。

VLC作为一款开源、免费、全能的多媒体播放器，在许多方面都有着强大的优势。虽然在界面和一些细节上可能不如一些商业软件那么精致，但其全面的功能和跨平台性使其在用户中享有广泛的好评。

MPC-HC（Media Player Classic – Home Cinema）

关键词：开源免费，精简高效，Windows党的福音

MPC-HC（Media Player Classic - Home Cinema） 是一款完全免费的开源软件。它的优点就是轻量化！占用资源极低，老爷机用户的福音。别看它小巧，功能却一点都不含糊，支持各种音视频格式，播放效果也堪称一流。

MPC-HC 的另一个亮点是高度可自定义。如果你是那种喜欢“折腾”的人，喜欢给播放器加点插件、调调滤镜，那它绝对会让你玩得不亦乐乎。不过，它的界面可能稍显老派，但对注重实用性的用户来说，这压根不是问题。

SMPlayer
关键词：跨平台，字幕搜索，YouTube播放

SMPlayer 是一款基于 MPlayer 和 MPV 引擎的播放器，它的亮点就在于“简单好用”。虽然界面极简，但功能一点也不简单！它内置了多种解码器，可以直接播放大多数音视频格式，还支持播放 YouTube 视频，追热点视频再也不用开浏览器了。

它还贴心地提供了字幕搜索和下载功能，甚至能自动记住你播放过的视频进度，这对追剧党来说简直太友好了。而且，它可以生成视频缩略图，方便你快速回顾视频内容，细节满分。

GOM Player

与 PotPlayer 比较类似，GOM Player 同样是来自韩国的本地视频播放器，支持播放 AVI，MP4，MKV，FLV，WMV，MOV 等几乎所有常用流行格式，甚至还支持线上搜索、下载解码器的功能。

GOM Player 以独有的技术支持在线传输中的文件、传送中断的文件、索引损坏的 AVI 文件的播放。

如果你既需要 PotPlayer 的简洁、强大和流畅，又需要干净、稳定等更多优良特性，那么可以试试 GOM Player。

Splash Player 支持运动补偿的播放器

GridPlayer 同一个界面同时播放多个视频播放器

GridPlayer 是一款基于 VLC 的免费开源跨平台多视频同步播放工具，支持在一块屏幕上同时播放多个视频。其主要功能包括：

多视频播放：用户可以在一个窗口中同时播放任意数量的视频，数量仅受硬件性能限制。
支持多种格式和流媒体：GridPlayer 支持所有由 VLC 支持的视频格式以及流媒体 URL（如 m3u8 链接）。
自定义网格布局：用户可以配置播放器的网格布局，以适应不同的观看需求。
硬件解码与软件解码：提供硬件解码和软件解码两种方式，以优化播放性能。
循环片段设置：用户可以设置具有帧精度的循环片段，方便重复观看特定部分的内容。
保存播放列表：用户可以保存播放列表，并保留有关位置、音量、循环、宽高比等信息。
拖放操作：通过拖放文件到播放器界面或使用右键菜单添加文件，实现快速播放。
此外，GridPlayer 还具备响应式设计，能够自适应各种屏幕尺寸，无论是桌面还是移动设备，都能提供一致的观看体验。它还支持 HTML5 视频元素和现代浏览器技术，确保流畅、高性能的视频播放。

总之，GridPlayer 是一款功能强大且灵活的多视频同步播放工具，适合需要同时观看多个视频的用户，如监控查看、视频比较等场景。

nPlayer (苹果ios端+安卓端)

nPlayer 是 iOS 上极受欢迎的一款万能格式局域网高清播放器 （也有安卓客户端）！

nPlayer的素质也确实没辜负它的盛名，除了可在 iPhone / iPad 和安卓上流畅播放几乎所有主流格式的视频、支持 DTS HD、Dolby AC3 / E-AC3 解码播放、外挂字幕等，还提供了强大的局域网远程播放能力；兼容性极佳，功能相当丰富。

Kodi（最重要的是可以在电脑上安装）

Kodi是一款开源的多媒体播放器，它可以在各种设备上安装，包括电视、电脑、平板电脑和智能手机等等。它可以播放多媒体文件，观看流媒体节目，并且还提供了强大的插件系统，通过众多的第三方插件可以实现很多的功能。

下载文件后，把文件拷贝到U盘中，再把U盘插入电视，然后直接安装。安装完毕后，进入Kodi，点击设置进行语言配置（电视无法截图，所以后面都以桌面端来演示，配置步骤一样）

进入设置界面后，选择Interface

选择Skin下的Fonts

在弹出界面选择Arial based字体（这一步不能漏，否则会乱码）

最后点击Regional下的Language

在弹出界面选择**Chinese(Simple)**，这样中文就设置好了

SMB配置
接着打开群晖，在控制面板下的文件服务中找到SMB协议，勾选启用SMB服务（如果要使用NFS、webDav协议访问群晖也是可以的，这里演示SMB）。SMB的高级设置不用更改，很多教程都提示改成SMB1，或者打开访客模式，都不安全

为了群晖的安全，在用户与群组下新建一个Kodi用户，只授予访问video文件夹的权限（电影资源在哪个文件夹就授予哪个权限）

新增的用户必须有配置访问SMB的权限，其他的无所谓

Kodi读取群晖资源
回到Kodi，现在需要为电影添加媒体源，也就是设置从哪里获取电影资源。在首页的电影选项中，点击进入文件区按钮

然后选择添加视频

点击右侧的浏览按钮

在弹出界面的最下面，选择添加网络位置

协议选择SMB，服务器名称输入群晖的IP地址，用户名和密码输入前面在群晖中创建的用户kodi和对应的密码

点击右侧的确定按钮后，就可以看到多出了群晖共享位置，点击后，就可以看到所有的群晖文件夹了

选择video文件夹后，直接点击右侧的确定，就代表选择了video文件夹作为电影的媒体源（其他文件夹是无法访问的，kodi用户没有访问其他文件夹的权限）

现在视频源中多了一个新的媒体路径，也可以在下面对媒体源进行重命名

然后可以在设置内容中设置目录包含和选择信息提供者（搜刮器）。点击右侧的确定按钮，就会扫描video文件夹中的所有资源了

扫描完成后，回到首页就可以在电影选项中看到video文件夹中的电影了

同样的，如果想要访问群晖中的电视剧，只需扫描电视剧存放的路径即可。所以建议把电视剧、电影分别存放在不同的文件夹中，方便后期扫描以建立索引

Kodi观看直播
下载直播源
看直播肯定需要直播源，去https://github.com/YueChan/Live下载m3u文件放进群晖中。这里建议登录上面的新建的kodi用户，把直播源放进File Station的home文件夹下，这样后面就可以方便的通过前面配置的SMB服务访问到群晖kodi用户下的直播源了

需要注意老宁下载的这个是IPv6的高清直播源，所以需要路由器开启IPv6才行，可以访问https://test-ipv6.com/来检测是否支持IPv6。m3u文件中，http后面有方括号的都是IPv6

需要注意的是，如果直播源的IPv6地址是80，必须要在IPv6的URL地址后面加上80端口，不能省略，否则kodi无法识别！所以IPv6直播源地址没有上面的:80，需要手动加上！

Kodi配置
回到Kodi中，在首页中进入插件选项，点击进入插件浏览

在新界面中直接点击搜索，输入IPTV Simple

搜出来PVR客户端 - IPTV Simple Client这个插件

安装完毕后进入插件设置

编辑默认的Config（或者添加一个附加配置）

进入常规选项卡，选择位置为本地路径，再点击播放列表路径

选择前面的群晖SMB地址，进入到home文件夹后，选择IPTV.m3u文件，点击确定

可以看到播放列表路径配置成功，最后点击右上角的确定

退回到首页，点击电视，稍微等一会

就可以选择频道进行观看了

配置节目单
节目单就是频道的界面表，可以不用进入频道就能看正在播放和将要播放的内容

需要进入前面安装的IPTV Simple Client插件，点击设置按钮，进入电子节目单中进行配置。位置要为远程路径，URL地址输入http://epg.51zmt.top:8000/e.xml，最后点击确定即可

现在回到电视中，就可以看到所有频道的节目单了

Kodi直接播放电视
Kodi的操作逻辑相对来说还是比较复杂的，菜单层层搞的老宁都是懵逼的，对于家里的老人来说更是复杂了。如果能在打开Kodi这个APP的时候直接播放电视，那对家人来说是最舒服的

进入Kodi的设置，点击界面

选择启动，点击启动时执行

选择播放电视节目

现在关闭Kodi后，再重新进入Kodi，那么就会自动播放电视节目了，爽不爽？现在还有个问题在换频道的时候，还要点击“确定键”，非常不方便，怎么办？

打开设置，进入PVR和电视直播配置

点击左下角，切换为专家模式

在播放选项中，关闭按确定确认频道切换选项。现在就可以直接按上下或者左右直接切换频道了

手机控制
Kodi提供了手机APP来方便控制Kodi客户端。首先去应用商店下载Kodi Remote。

下载完后打开APP，点击左下角的添加站点，按界面的提示，需要在Kodi中激活远程控制。在Kodi中点击设置进到控制界面，需要先设置用户名和密码才能打开HTTP远程控制

配置完成后，在APP首页中点击添加站点，点击查找Kodi，输入用户密码后，再点击保存

最后点击需要控制的Kodi客户端就可以了

点击左上击的三条线打开侧边栏，可以看到和Kodi客户端上菜单的基本一致，是可以直接使用APP进行控制的。最重要是的APP提供了遥控器功能

打开后，可以直接用手机遥控Kodi了，如果你是按照文章的顺序来操作的，那就知道用电视遥控器输入一个URL地址是多么崩溃的事！有了它可以直接用手机键盘输入复杂的URL地址，而不用电视遥控器了

Mx Player（安卓端）

MX Player是我自用多年的安卓本地视频播放器，它强悍的功能，让我觉得它就是安卓手机上欣赏影片的最佳途径，没有之一。

今天，我把它推荐给大家，希望它也可以让你用手机观影的体验提升。

MX Player的强大源于它拥有现如今最新的硬件解码器，使得视频播放更加流畅，

它的多核心的解码也要比单核心的解码效率高出70%，

而且，它还可以在线匹配字幕，免去因为影视资源没有字幕，而自己花时间去找字幕的麻烦。

IINA

IINA 是一款专为 macOS 设计的开源视频播放器，以其优雅的界面、强大的格式支持和无广告的特性受到用户青睐。它支持暗黑模式、触控板手势、Touch Bar 和画中画模式，提供类似苹果官方播放器的体验。IINA 基于 mpv 内核，几乎可以播放所有视频和音频格式，包括 mkv、flac 等。此外，它还具备强大的自定义能力，如调整播放速度、更换主题、调节 UI 透明度等。IINA 支持在线播放和 AirPlay 投屏，无需下载即可享受大屏观影体验。与 VLC 和 QuickTime 相比，IINA 在 macOS 设计、格式支持、UI 美观度、触控手势、在线播放和插件扩展方面表现更优。IINA 的安装简便，用户可通过官网或 GitHub 下载，或使用 Homebrew 安装。对于追求高颜值、良好体验和强大功能的 Mac 用户来说，IINA 是一个值得尝试的免费且无广告的播放器选择。
